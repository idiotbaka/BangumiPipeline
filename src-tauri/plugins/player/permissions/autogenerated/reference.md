## Default Permission

Allows the mobile player to enter and exit native fullscreen.

#### This default permission set includes the following:

- `allow-enterFullscreen`
- `allow-exitFullscreen`
- `allow-enter-fullscreen`
- `allow-exit-fullscreen`
