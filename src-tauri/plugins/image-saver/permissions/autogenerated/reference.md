## Default Permission

Allows BakaVip2 to prepare images and save them to the public Pictures folder.

#### This default permission set includes the following:

- `allow-prepareImage`
- `allow-saveImage`
- `allow-discardImage`
- `allow-prepare-image`
- `allow-save-image`
- `allow-discard-image`
